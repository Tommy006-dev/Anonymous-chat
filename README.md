## Installation & Setup Guide

Follow these steps to run the project locally on your machine.

### Prerequisites
* Ensure you have [Node.js](https://nodejs.org/) installed (v14.x or higher recommended).

### Step 1: Install Dependencies
Open your terminal (Command Prompt, PowerShell, or VS Code Terminal), navigate to the root directory of this project, and run this command:
npm install express socket.io cors
### Step 2: Start the Backend Server
run this command:
 node server.js to start server
### Step 3: Open your browser and go to http://localhost:3000.
